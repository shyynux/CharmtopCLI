## Features

[`Version 1`] Organizes all the screenshots into a folder.

## How to use this?

Open terminal and navigate to your desktop directory.

Install CharmtopCLI:
```bash
npm install charmtopcli
```
Check if it's installed: 
```bash
charmtopcli
```

Organize your desktop: 
```bash
charmtopcli charm
```
